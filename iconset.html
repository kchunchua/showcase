Animated Icon Set

This is my experiment of interactive icons creation which can be used in any platforms (Website / App) with a small file and speed load. This set of animated icons features toggling function. This animated icons set is implemented by using the Lottie/BodyMovin which is a superpower plugin developed by AirBnB. I used After Effect to create animation scenes for each toggling action. A simple and clean Javascript sniplet is triggered the animation scene to play the exported JSON file smoothly.
